run_started

Framing event — first line of every trace file. Stamps trace_version, run_id, agent_name, task_count.

{
  "trace_version": 1,
  "kind": "run_started",
  "run_id": "1c0fa8e7d4ad",
  "agent_name": "causal",
  "task_count": 480,
  "seed": 42,
  "timestamp": 1748557812.41
}
step

One per agent iteration. Carries full ranking + chosen action + adjustment set + outcome — the unit the auditor & explainability rubric ingest.

{
  "kind": "step",
  "task_id": "swe-bench/abc123",
  "step_index": 2,
  "history_size": 38,
  "chosen_action": "run_tests",
  "scores": [
    {"action": "run_tests",
     "score": 0.73,
     "rationale": "via back-door {…}",
     "adjustment_set": ["…"]}
  ],
  "outcome": 0.62,
  "reflection": "appended synthetic row(s)",
  "elapsed_seconds": 1.84
}
task_result & run_finished

Per-task summary, then a single closing event. Auditor counts the closing event to detect a clean run vs a crashed one.

{
  "kind": "task_result",
  "task_id": "swe-bench/abc123",
  "outcome": "solved",
  "final_outcome_score": 1.0,
  "n_steps": 3,
  "error": null
}
{ "kind": "run_finished",
  "run_id": "1c0fa8e7d4ad" }
