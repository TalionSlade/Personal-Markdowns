Path annotations

load
JSONL loader assigns 4 axis labels per task — language, framework, bug_type, codebase_size.
partition
SplitPartitioner returns three buckets; runner aggregates metrics per bucket.
decide
Per step: scorer reads history, ranks candidates, agent picks argmax.
act
Tool invoked with frozen state snapshot; output records mediators & outcome.
reflect
CRM compares predicted vs observed outcome; if δ > θ, append synthetic rows.
emit
RunObserver hands each step + result to TraceWriter (or NullTraceWriter).
audit
TraceAudit counts cold-start vs steady-state vs reflection-triggered steps.
report
EvaluationMetrics + BCa CI + paired permutation test → dissertation table.
