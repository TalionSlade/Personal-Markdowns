BCa Bootstrap & Paired Permutation Tests

Distribution-free uncertainty estimates with no SciPy dependency

Causa · §K2
BCa bootstrap CI for the mean
Bias-corrected, accelerated bootstrap — Acklam normal-inverse approximation keeps the implementation SciPy-free.
1
Draw B bootstrap resamples of size n (default n_bootstrap=2000) from the observation sample.
2
Compute the statistic (mean) on each resample → boot_means.
3
Bias correction z0 from the fraction of boot_means < θ_hat.
4
Acceleration a from jackknife skewness — handles non-symmetric distributions.
5
Adjusted percentiles α1, α2 via the normal-inverse approximation; return the quantiles.
Acceleration
a = Σ (θ̄ − θ_i)³ / [6 · (Σ (θ̄ − θ_i)²)^{1.5}]  
Adjusted quantile
α_k = Φ(z0 + (z0 + z_k) / (1 − a (z0 + z_k)))
Invariants — n ≥ 2 required; seed reproducible; lower < point < upper always; the CI contains the true mean ≥ 95% of the time for high-n samples (covered by test_stats).
Paired permutation test (sign-flip)
Exhaustive enumeration for small n; Monte Carlo with Phipson-Smyth correction above.
1
Compute observed mean difference θ̂ = mean(a) − mean(b).
2
If n ≤ 14: enumerate all 2n sign-flip vectors of (a − b); count those with |Δ| ≥ |θ̂|.
3
Else: draw n_permutations random sign-flips (default 10,000).
4
Apply +1 Phipson-Smyth correction for Monte Carlo: p = (k+1) / (B+1).
5
Return PermutationResult(method, mean_diff, p_value).
Exact regime
p = #{ ε ∈ {±1}^n : |mean(ε ⊙ d)| ≥ |θ̂| } / 2^n
Monte Carlo regime
p̂ = (1 + #{ε_b : |mean(ε_b ⊙ d)| ≥ |θ̂|}) / (1 + B)
Why Phipson-Smyth? Naive p̂ = k/B can equal 0 for finite B; the +1 correction guarantees a strictly positive lower bound (so −log p is finite). Required for valid downstream meta-analysis.
