Causal Graph Specification

A typed, DAG-validated representation of the SCM. Hand-authored for the debugging domain; LLM-extracted for novel domains.

Code: causa.core.graph, causa.core.scm, causa.domain.scm_debugging, causa.extraction.llm_extractor.
hand-authored DAG LLM-extracted DAG cycle filtering
§C2
2

Identifiable Action Scorer

Pearl's back-door criterion + a plug-in linear estimator. Falls back to LLM cold-start when data is sparse.

Code: causa.core.identifiability, causa.adapters.estimators.linear_regression, causa.planning.dowhy_scorer, causa.planning.llm_scorer.
do-calculus DoWhy adapter LLM cold-start
§D2
3

Causal Planning Agent

The decision loop that consumes scorer rankings, invokes tools, accumulates SCM-shaped observations, and reports a typed result.

Code: causa.agents.base.BaseAgent, causa.agents.causal.CausalPlanningAgent, causa.adapters.history.pandas_history, causa.adapters.tools.
decision loop step trace success threshold
§E1
4

Counterfactual Reflection Module

Detects discrepancies between SCM predictions and observed outcomes; injects LLM-derived synthetic data to keep the SCM honest under shift.

Code: causa.reflection.module.CounterfactualReflectionModule, causa.adapters.threshold.{StaticThreshold,AdaptiveMedianThreshold}.
discrepancy adaptive θ synthetic rows
